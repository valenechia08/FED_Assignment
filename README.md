Existing features

User Account Management
Feature 1 : Users can choose to log in as a patron, vendor or NEA officer.

Feature 2 : Vendors can create accounts to manage stalls, menus, and orders, patrons can also create accounts or log in as a guest to order food.

Feature 3 :  Vendors, patrons and NEA officers can reset their passwords if they forget it which includes a finding their account page, a security verification page to ensure that only the respective users can log into their accounts and a second security verification if users did not receive an OTP, a reset their password page and lastly, a page showing that their password reset was successful.

Ordering & Checkout
Feature 1 : Patrons can view the stalls in the homepage.

Feature 2 : Patrons can filter stalls based on cuisine. (Note: Click on the “Chinese” picture)

Feature 3 : Patrons can search for stalls using the search bar.

Feature 4 : Patrons can click on stalls and view their menus. (Note: Only “Banana Leaf Nasi Lemak” & “Boon Lay Fried Carrot Cake & Kway Teow Mee” can be clicked right now and the heart icons for both mobile and desktop view were added in by Aadya)

Feature 5 : Patrons can add items into their cart and remove them.

Feature 6 : Patrons can checkout, choose their preferred service. (i.e. self-pickup or dine-in) and payment success or failure will be displayed.
